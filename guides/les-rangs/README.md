# Les Rangs

## Introduction

Le serveur Orezia propose **différents rangs**, tous différents les uns que les autres, et tous ont leurs utilités.

Il existe 2 grandes familles de rangs :

> **Rang Joueur** : Rang classique, et principale hiérarchie. Plus vous montez, plus vous avez des avantages. Vous commencez au rang de Survivant à l'arrivée sur le serveur.\
> **Grade Premium** : Grade boutique, qui vous permet de gagner en avantages sur le serveur.
